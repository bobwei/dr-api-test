### Installation

Using [npm](https://www.npmjs.com/):

    $ npm install
    $ npm install -g babel-node


Create your .env file

```
api_key=
redirect_uri=
access_token=
```

P.S.

Please leave "access_token" field blank and update it after autorization step.


### Run

    $ npm start
